## 五、有意保留的限制项（后续可做）

**主题设置的 light/dark 不作用于主搜索窗**（仅作用于管理中心）。原因：主搜索窗是透明窗，背景是 Rust 侧固定浅色 Acrylic 底（`window.rs` TINT=246,246,248）；若只改 CSS 强制深色，浅色文字压在浅色 Acrylic 上会几乎不可读。真正的深色需让原生 Acrylic 底色随主题动态切换（改 `window.rs` 的 TINT/alpha 逻辑并在 save_settings 时对主窗口重应用），属较大改动，故本次保留，主窗口仍跟随系统外观。
