# 系统功能彩色图标 — 交接文档

> 状态：**已完成并构建验证**（tsc 通过 / debug exe 构建成功 / dist 产物已含新图标）。
> 本文记录：图标来源、最终成果、**可复用的自动化切图流程与脚本**、踩过的坑、以及新增图标的 SOP。

## 一、背景

iTools 主搜索面板的系统功能（如「卸载或更新程序」「电源选项」「环境变量」「资源监视器」等）与内置工具原本大量缺图标，只能显示系统提取的灰度小图或占位。目标：给它们配上 **Windows 11 Fluent 风格的彩色圆角方块图标**，风格统一。

**图标来源决策（海风哥拍板）**：用大模型（ChatGPT / DALL·E）生成，不手绘 SVG——「gpt 生成不会有版权问题，它生成好看」。

## 二、最终成果

`src/system-icons.ts` 现含 **42 个系统功能彩色图标**（base64 PNG data URL，96×96 透明方块）：

- **原 12 个**（海风哥早期手工切片 `doc/icons/1-12.png`）：卸载或更新程序、控制面板、设备管理器、服务、任务管理器、注册表编辑器、磁盘管理、网络连接、电源选项、系统属性、系统设置、Windows 更新。

## 三、自动化切图流程（可复用）

大模型**生成不了真透明图**——即使 prompt 要求"透明背景"，实际下载得到的是 **24bpp 白底不透明图**（背景 RGB≈245-254）。所以流程是「生成纯色底 → 本地去背景 → 切」：

1. **生成**：让 GPT 生成一张网格图（如 3 列 × 4 行 = 12 个图标），**明确要求纯色背景**（白/黑/绿等，按图标内容选，方便抠图），圆角彩色方块 + 白色简洁符号，Windows 11 Fluent 风格，大小完全一致、居中对齐；prompt 里**逐个编号说明每格画什么、什么颜色**。
2. **下载**：浏览器 canvas 导出，或点图的分享面板「下载」。`.crdownload` 即完整文件（Chrome 只是挂在"保存"确认里），可直接读取处理。
3. **去背景 + 切 + 命名**：跑下面的 PowerShell 脚本。
4. **集成**：base64 追加进 `system-icons.ts`。
5. **构建**：`npm run tauri build -- --debug --no-bundle`。

### 核心切图脚本（去白底 + 连通区域切 + 行优先命名）

```powershell
Add-Type -AssemblyName System.Drawing
# ==== 按需修改 ====
$path  = "C:\path\to\grid.png"   # 源网格图（白底不透明）
$cols  = 4; $rows = 3            # 网格列数/行数
$names = @('图标1','图标2', ...) # 行优先顺序，个数 = cols*rows，须与后端展示名一致
$out   = "F:\ClaudeWorkspaces\itools\tmp_icons"
# =================
New-Item -ItemType Directory -Force -Path $out | Out-Null
$src24 = New-Object System.Drawing.Bitmap($path); $W=$src24.Width; $H=$src24.Height
$bmp = New-Object System.Drawing.Bitmap($W,$H,[System.Drawing.Imaging.PixelFormat]::Format32bppArgb)
$g0=[System.Drawing.Graphics]::FromImage($bmp); $g0.DrawImage($src24,0,0,$W,$H); $g0.Dispose(); $src24.Dispose()
$bd=$bmp.LockBits((New-Object System.Drawing.Rectangle(0,0,$W,$H)),[System.Drawing.Imaging.ImageLockMode]::ReadWrite,[System.Drawing.Imaging.PixelFormat]::Format32bppArgb)
$stride=$bd.Stride; $buf=New-Object byte[] ($stride*$H); [System.Runtime.InteropServices.Marshal]::Copy($bd.Scan0,$buf,0,$buf.Length); $N=$W*$H
# ① 整图去白底：四角 flood-fill，brightness>235 的边缘连通像素 alpha 置 0（图标内部白符号被彩色包住，不受影响）
$vis=New-Object bool[] $N; $st=New-Object System.Collections.Generic.Stack[int]; $st.Push(0);$st.Push($W-1);$st.Push(($H-1)*$W);$st.Push($N-1)
while($st.Count -gt 0){ $v=$st.Pop(); if($v -lt 0 -or $v -ge $N -or $vis[$v]){continue}; $vis[$v]=$true; $px=$v%$W;$py=[int]($v/$W);$o=$py*$stride+$px*4
  if((0.299*$buf[$o+2]+0.587*$buf[$o+1]+0.114*$buf[$o]) -lt 235){continue}; $buf[$o+3]=0
  if($px -gt 0){$st.Push($v-1)};if($px -lt $W-1){$st.Push($v+1)};if($py -gt 0){$st.Push($v-$W)};if($py -lt $H-1){$st.Push($v+$W)} }
[System.Runtime.InteropServices.Marshal]::Copy($buf,0,$bd.Scan0,$buf.Length); $bmp.UnlockBits($bd)
# ② 连通区域找每个图标色块（alpha>128）
$lab=New-Object bool[] $N; $blobs=New-Object System.Collections.Generic.List[object]
for($i=0;$i -lt $N;$i++){ if($lab[$i]){continue}; $o0=[int]($i/$W)*$stride+($i%$W)*4; if($buf[$o0+3] -le 128){$lab[$i]=$true;continue}
  $mnX=$W;$mxX=0;$mnY=$H;$mxY=0;$cnt=0; $stk=New-Object System.Collections.Generic.Stack[int]; $stk.Push($i)
  while($stk.Count -gt 0){ $v=$stk.Pop(); if($v -lt 0 -or $v -ge $N -or $lab[$v]){continue}; $px=$v%$W;$py=[int]($v/$W);$o=$py*$stride+$px*4
    if($buf[$o+3] -le 128){$lab[$v]=$true;continue}; $lab[$v]=$true;$cnt++
    if($px -lt $mnX){$mnX=$px};if($px -gt $mxX){$mxX=$px};if($py -lt $mnY){$mnY=$py};if($py -gt $mxY){$mxY=$py}
    if($px -gt 0){$stk.Push($v-1)};if($px -lt $W-1){$stk.Push($v+1)};if($py -gt 0){$stk.Push($v-$W)};if($py -lt $H-1){$stk.Push($v+$W)} }
  if($cnt -gt 3000){ $blobs.Add([PSCustomObject]@{mnX=$mnX;mnY=$mnY;mxX=$mxX;mxY=$mxY;cx=($mnX+$mxX)/2;cy=($mnY+$mxY)/2}) } }
# ③ 行优先排序（务必用 Math::Floor，PowerShell 的 [int] 是四舍五入！）
$bandH=$H/[double]$rows
$sorted=@($blobs | Sort-Object @{e={[Math]::Floor($_.cy/$bandH)}}, @{e={$_.cx}})
# ④ 裁 bbox 填 96×96 正方形，按名保存
for($idx=0;$idx -lt $sorted.Count -and $idx -lt $names.Count;$idx++){ $bl=$sorted[$idx]; $bw=$bl.mxX-$bl.mnX+1;$bh=$bl.mxY-$bl.mnY+1
  $o2=[System.Drawing.Bitmap]::new(96,96); $gr=[System.Drawing.Graphics]::FromImage($o2); $gr.InterpolationMode='HighQualityBicubic'; $gr.Clear([System.Drawing.Color]::Transparent)
  $gr.DrawImage($bmp,(New-Object System.Drawing.Rectangle(0,0,96,96)),$bl.mnX,$bl.mnY,$bw,$bh,[System.Drawing.GraphicsUnit]::Pixel); $gr.Dispose()
  $o2.Save((Join-Path $out ($names[$idx]+".png")),[System.Drawing.Imaging.ImageFormat]::Png); $o2.Dispose() }
$bmp.Dispose(); "切出 $($sorted.Count)/$($names.Count) 个"
```

### 追加进 system-icons.ts

读取每个 `名字.png` → `[Convert]::ToBase64String(...)` → 拼成 `  "名字": "data:image/png;base64,...",` → 插到 `system-icons.ts` 收尾 `}` 之前（保证上一条目结尾有逗号）。写文件用 `[Text.UTF8Encoding]::new($false)`（无 BOM）。

## 四、踩过的坑（重要）

1. **大模型生不了真透明**：prompt 写"透明背景"没用，下载得到 24bpp 白底不透明图（Read 显示的棋盘格是渲染出来的假象）。→ 要纯色底 + 本地 flood-fill 去背景。（已记 engram L2）
2. **PowerShell `[int]` 是四舍五入不是截断**：`[int](3/4)=1`。行分组、网格排版一律用 `[Math]::Floor`，否则图标/名字错位——本次排查耗时最多的假象就源于此（切图其实一直是对的，是预览渲染的 [int] 把图标画歪了）。
3. **`New-Object Bitmap($a,$b)` 括号逗号语法**会被当成"一个数组参数"，报 `op_Addition`/null。改用 `[System.Drawing.Bitmap]::new($a,$b)`。
4. **硬切网格（按 1/cols 均分）不可靠**：图标间距不均、会跨格切错。必须用**连通区域检测**按每个色块实际边界切。
5. `Remove-Item` 对刚建的空目录用 `*` 通配会触发"保护路径"拦截——删具体文件/目录名即可。

## 五、新增图标 SOP

1. 后端 `apps.rs::add_system_commands` 加命令（展示名、目标）。
2. GPT 生成纯色底网格图（prompt 逐格编号说明），下载。
3. 改第三节脚本的 `$path/$cols/$rows/$names`（`$names` 行优先、与展示名一致）跑一遍，肉眼核对切图。
4. base64 追加进 `system-icons.ts`。
5. `npx tsc --noEmit` → `npm run tauri build -- --debug --no-bundle`。
